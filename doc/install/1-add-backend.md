#### Install a plugin

```bash
pnpm add -D @nrwl/express
```

#### Add a simple `express` backend

```bash
pnpm nx g @nrwl/express:app backend
```

#### Install the GraphQl packages

```bash
pnpm add @apollo/client
pnpm add graphql
pnpm add express-graphql
```

#### Add Cors

```bash
pnpm add cors
```
